## ðŸ“ˆ Progression System

### Piece Upgrades
- **Unlock**: Purchase new pieces with coins
- **Level Up**: 5 levels per piece (+20% stats each)
- **Promotion**: Level 3+ pieces can become "Super" versions (double damage)

### Skill Tree
- **Faster Reload**: Reduces weapon cooldown (15% per level, max 45%)
- **More Health**: Extra lives at game start (+1 per level, max +3)
- **Bonus XP**: Increases XP gain (20% per level, max 60%)

### Daily Challenges
Reset every 24 hours:
- **Score Challenge**: Reach target score
- **Kill Challenge**: Eliminate X enemies
- **Survival Challenge**: Survive X seconds

### Currency System
- **XP**: Gained from kills, levels up for coin rewards
- **Coins**: Used for upgrades and unlocks
- **Levels**: Unlock at 100 XP per level
